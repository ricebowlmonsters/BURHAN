Mode Offline: Data akan disimpan di browser komputer ini (Local Storage). Tidak membutuhkan internet, tapi data tidak bisa diakses dari perangkat lain.

Mode Server PC: Data disimpan di file komputer.

• Satu PC: Gunakan http://localhost:3001/db
• Beda Device (Satu WiFi): Gunakan IP Komputer (Cek terminal server, cth: http://192.168.1.5:3001/db)

⚠️ Peringatan: Anda sedang mengakses web ini secara Online (Domain/HTTPS). Koneksi ke "Localhost" mungkin akan diblokir oleh browser. Disarankan menggunakan Firebase untuk mode Online.
