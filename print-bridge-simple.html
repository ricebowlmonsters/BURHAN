🖨️ Print Bridge - Thermal Printer via LAN

Penjelasan Singkat:
File ini menjelaskan cara menggunakan print bridge untuk auto-print ke printer thermal via LAN tanpa dialog print.

Metode 1: Menggunakan Browser (Paling Mudah)

Untuk saat ini, sistem sudah bekerja dengan browser. Auto-print akan muncul dialog print, dan Anda hanya perlu klik OK.

Tips: Set default printer di Windows ke printer thermal utama, maka dialog print akan langsung ke printer tersebut.

Metode 2: Menggunakan Print Bridge Script (Auto-print Penuh)

Jika Anda ingin benar-benar auto-print tanpa dialog sama sekali:

Langkah 1: Install Node.js

  1. Download Node.js dari nodejs.org
  2. Install seperti aplikasi biasa
  3. Buka Command Prompt / Terminal
  4. Test: ketik node --version (harus muncul versi)

Langkah 2: Setup Print Bridge

# Masuk ke folder aplikasi kasir cd "C:\Users\RUFINA\Documents\Procejet kasir\wep app kasir" # Install dependencies npm install express body-parser # Edit file print-bridge.js, ganti IP printer sesuai printer Anda: # PRINTER_CONFIG = { # 'MAKANAN': '192.168.1.100', # IP printer MAKANAN # 'MINUMAN': '192.168.1.101', # IP printer MINUMAN # 'KASIR': '192.168.1.102', # IP printer KASIR # } # Jalankan print bridge node print-bridge.js

Langkah 3: Update Web App untuk Menggunakan Bridge

Web app perlu dikonfigurasi untuk mengirim request ke print bridge. (Perlu modifikasi kode JavaScript)

Metode 3: Menggunakan Electron App (Paling Lengkap)

Membuat aplikasi desktop yang bisa auto-print tanpa dialog. Ini membutuhkan lebih banyak setup, tapi hasilnya aplikasi desktop yang standalone.

Catatan: Metode ini lebih kompleks dan membutuhkan pengetahuan programming lebih lanjut.

Rekomendasi untuk Anda:
Jika printer sudah dikonfigurasi sebagai default printer di Windows, maka penggunaan browser biasa sudah cukup. Dialog print akan muncul, tapi cukup klik OK sekali, dan browser akan ingat pilihannya.
Penting: Pastikan printer thermal Anda support raw TCP/IP printing pada port 9100. Kebanyakan printer thermal modern sudah support ini.

Konfigurasi Printer di Windows (Untuk Browser Mode)

  1. Pastikan printer thermal sudah terinstall di Windows
  2. Buka Settings → Devices → Printers & scanners
  3. Klik printer thermal yang ingin jadi default
  4. Klik Manage → Set as default
  5. Sekarang ketika auto-print, printer ini akan terpilih otomatis
